MyAnimeList

Application for IPB4
Developed by Magnum357

Introduction

What is it?a bbcode that makes a box

Pulls the data from MyAnimeList according to the desired values and create an info box.

Installation

First step: Applicationthis does everything
  1. Go to applications list: ACP / System Tab / Site Features Menu Item / Applications Sub Menu Item.
  2. Click on Install and select the application file (with tar extension).
  3. Start the installation by clicking Install on the pop up window.


Second and last step: Editor Pluginto create the bbcode quickly
  1. Go to the page to edit the editor: ACP / Customizastion Tab / Editor Menu Item / Toolbar Sub Menu Item.
  2. Click on Install Button and select the editor file (with zip extension).
  3. Start the installation by clicking Save on the new window.
  4. Hold the MyAnimeList button/icon on the buttons not on editor section and drag it to the above editor.

Usage

BBCode Syntaxlink parameters

Get the required values from the link you want to create an info box:

https://myanimelist.net/anime/5114/Fullmetal_Alchemist__Brotherhood

And put the values to this:

[mal type=anime id=5114]

All you need to do. After you post it, the info box will be created.
